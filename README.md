# imc-app
# Calculadora IMC

Aplicación de escritorio desarrollada con Electron para calcular el Índice de Masa Corporal (IMC) y mostrar un historial de cálculos guardados localmente en un archivo JSON.

## Funcionalidades
- Captura de nombre, peso y altura
- Cálculo automático del IMC
- Clasificación del resultado:
  - Bajo peso
  - Peso normal
  - Sobrepeso
  - Obesidad
- Historial persistente de cálculos

## Tecnologías utilizadas
- Electron
- HTML
- CSS
- JavaScript
- JSON para persistencia de datos

## Cómo ejecutar el proyecto
1. Instalar dependencias:
   npm install

2. Iniciar la aplicación:
   npm start
